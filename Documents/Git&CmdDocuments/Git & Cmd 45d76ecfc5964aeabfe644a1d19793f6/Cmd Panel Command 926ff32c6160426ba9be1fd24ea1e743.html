Cls Command

cls komutu paneli tamamen temizler

cd.. Command

  • cd.. komutu bulunduğun dizinden bir önceki klasöre geçer.

cd 'girmek istediğin klasör ismi'

  • cd ‘file name’ Seçili klasör dizinine girer klasör ismi küçük büyük farketmez tab'a basarak ilgili klasörleri otamatik gösterir.

dir

  • dir komutu içinde bulunduğun dizinin tüm dosyalarını gösterir.

mkdir

  • mkdir veya md komutu klasör oluşturmamızı sağlıyor.
  • mkdir a,b,c ~> a,b ve c adında 3 klasör oluşturduk.

rmdir

  • rmdir veya rd komutu klasör silmeye yarıyor.
  • rmdir a,b,c ~> a,b ve c adındaki 3 klasörü sildik. !!! amma ve lakin içi dolu olan bir klasörü bu şekilde silemiyoruz.
  • rmdir /S 'file name’ içi dolu olan bir klasörü silmek için /S paramatresi kullanılmalı.

move

  • move komutu seçili dosyayı kesip belirlenen alana taşır.
  • move a b ~>a dosyasını aynı dizinde bulunan b klasörüne taşıyoruz.
  • move a D:/b ~> a dosyasını D diskinde bulunan b klasörüne taşıyoruz.

copy

  • copy komutu seçili dosyayı kopyalayıp belirlenen alana taşır.

rename

  • rename komutu seçili dosyanın ismini değiştirir.
  • rename a b ~> a dosyasının ismini b olarak değiştirir.

del

  • del komutu (dosya / belge / foto) silmeyi sağlıyor.
  • del a.jpg,b.jpg,c.jpg ~> a,b ve c resimleirini siler.

touch

  • touch komutu metin dosyası oluşturur
  • touch dosya.txt ~> dosya isminde metin dosyası oluşturur.

echo

  • echo komutu panele veya istediğiniz metin dosyasının içine string değer yazmamızı sağlar.
  • echo "Hello World" > dosya.txt ~> dosya.txt metin dosyasının içine “Helllo World” yazdırır.
  • büyüktür işareti soldaki şeyi sağdaki dosyanın içine yazdırır.
  • systeminfo > sistembilgileri.txt
  • dersek sistem bilgilerini sistem bilgisi adlı not defterine kaydetmiş oluruz.

type

  • type komutu seçili dosyanın dosyanın içeriğini yazar.
  • type "not.txt" ~>not defteri metin dosyasının içeriğini yazar.

find

  • find komutu dosya içinde string değer arar.
  • find /n "Hello World" dosya.txt ~> dosya.txt metin dosyasının içerisinde “Helllo World” yazısını arar.

DRİVEQUERY

  • DRİVEQUERY komutu(büyük harflerle) bilgisayara yüklü sürücüleri gösterir.

ipconfig

  • ipconfig komutu ip bilgilerini gösterir
  • ipconfig /all ~> detaylı ip bilgilerini veriyor

code .

  • code . komutu dosya dizinini visual studio code da açar. misal: C:\users\desktop\Monster Notebook\PrograminLanguange\KitabıOyla code .

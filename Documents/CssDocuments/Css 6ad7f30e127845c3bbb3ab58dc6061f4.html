gap

Kapsayıcı elemente gap değeri atadığımız zaman alt elementler ızgara sistemine girer. Köşeye yakın etiketler hariç alt elementin herbiri eşit derece boşluk değeri alır.
  • row-gap satırlar arası boşluk.
  • column-gap sutunlar arası boşluk.
  • gap: satır ve sutun arası boşluk row-gap ve column-gap tek satır yazılışı.
.container{
display:flex;
row-gap:1em;  /* satırlar arası 1em boşluk bırakır */
}
.container{
display:flex;
column-gap:1em;  /* sutunlar arası 1em boşluk bırakır */
}
.container{
display:flex;
gap:1em;        /* sutun ve satırlar arası 1em boşluk bırakır */
gap:1em 2em;    /* satırlar arası 1em sutunlar arası 2em boşluk bırakır */
}

overflow

Kutudan taşan yerlere ne olucağını seçtiğimiz bir özellik
  • overflow → tüm taşan kısımlar
  • overflow-x → sağa sola taşan kısımlar
  • overflow-y → yukarı aşağı taşan kısımlar
overflow: visible;  
/* visible taşan yerleri görünür yapar 
ve overflow'un varsayılan değer visibledır */

overflow: hidden;
/* hidden taşan yerleri gizler ulaşılamaz yapar */

overflow: scrool;
/* scrool taşan yerleri gizler 
ve scrool ekler bu sayede gizli yerlere erişilebilir*/

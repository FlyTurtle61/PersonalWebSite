Selectors

  • h1 p → h1 in içindeki tüm p leri seçer
  • h1 + p → h1 etiketinde sonraki ilk gelen p’yi seçer
  • h1 > p → sadece h1 tagının içindeki p leri seçer
  • h1 ~ p → h1 başlığından sonra gelen tüm p leri seçer

CLAS

Birden fazla tagı aynı css özellikleri alması için kullanabiliriz.
  • html taglarını dilediğimiz gibi özelleştirebiliyoruz.
  • Classlar css’de .(özel_isim) şeklinde kullanılır.
<div>
	<p class="paragraph">HELE gardaş</p>
</div>

bunu css de belirtirken

.paragraph p
{ 
 color:black;
}

ID

Clasdan iki farkı var onlarda:
  • Her bir id sadece 1 tag için kullanılabilir.
  • Id’ler css’de #(özel_isim) şeklinde kullanılır.
<div>
	<p id="paragraph">HELE gardaş</p>
</div>

bunu css de belirtirken

#paragraph p
{ 
 color:black;
}

::first-letter

::first-letter özelliği ilgili tagın ilk harfini seçer.
p:first-letter{
 /* her paragrafın ilk harfi */
color:white;
line-height:25px;
text-transform:capitalize;
}

:first-line

::first-line özelliği paragrafın ilk satırının tamamını seçer.
p:first-line{	
/* her paragrafın ilk satırının tamamı */
}

:nth-child()

p:nth-child(n) → üst öğenin n’inci p tagı
  • Html
<body>

		<p>I am a strong man.</p>
		<p>I am a strong man.</p>

</body>
  • Css
p:nth-child(2)
{
	color:red;    /* 2. p tagının text rengini kırmızı yaptık */
}

:first-child

p:first-child → üst öğenin birinci p tagını seçer
  • Html
<body>

		<p>I am a strong man.</p>
		<p>I am a strong man.</p>

</body>
  • Css
p:first-child
{
	color:red;    /* birinci p tagının text rengini kırmızı yaptık */
}

:last-child

p:last-child → üst öğenin sonuncu p tagını seçer
  • Html
<body>

		<p>I am a strong man.</p>
		<p>I am a strong man.</p>

</body>
  • Css
p:last-child
{
	color:red;    /* sonuncu p tagının text rengini kırmızı yaptık */
}

::before

::before ilgili tagın soluna bir takım işlemler yapmamızı sağlar
p:before{
content:url(resim.jpg);  /* p tagının soluna resim koyar */
width:100%;
height:auto;
}
p:after{
content:'Yazı yazıyorum';  /* p tagının sağına yazı yazıyorumu yazar */
width:100%;
height:auto;
}

::after

::after ilgili tagın sağına abir takım işlemler yapmamızı sağlar
p:before{
content:'Yazı yazıyorum';  /* p tagının soluna yazı yazıyorumu yazar */
width:100%;
height:auto;
}

p:after{
content:url(resim.jpg);  /* p tagının sağına resim ekler */
width:100%;
height:auto;
}

:hover

:hover fare imleci ilgili tagın üzerine gelince çalışan css kodu
a:hover{
/* mouse linkin üstüne gelince */
color:black;
font-size:20px;
}

:active

:active fare imleci ilgili taga tıkladığı zaman çalışan css kodu
a:active{
/* farenin linke bastığı an */
color:blue;
}

:visited

:visitied fare imleci ilgili taga daha önceden tıkladıysa çalışan css kodu
a:visited{
/* fare linke bastýktan sonra ziyaret edilmiþ site */ 
color:purple;
}

:focus-within

Tıklandığında
  • Bir input focus olduğunda ona erişmek için :focus-within kullanırız.
input[type=text]:focus-within
{
	border-color:gray;
}

:invalid

geçersiz
  • Bir input geçersiz olduğunda ona erişmek için :invalid kullanırız.
input[type=mail]:invalid
{
	border-color:red;
	transform: scale(1.1);
}
input:invalid 
{
	border-color:red;
}
/*  geçersiz input */

:valid

geçerli
  • Bir input geçerli olduğunda ona erişmek için :validkullanırız.
input[type=mail]:valid
{
	border-color:green;
	transform: scale(1.1);
}
input:valid
{
	border-color:green;
}
/*  geçerli değer input */

:-moz-input-placeholder { /* Mozilla Firefox */

::placeholder {

}

:target

Site içi geçiş için a tagı kullandık ve bastığımız zaman gittiğimiz hedefi seçmek için :target kullanılır.
<h1>This is a heading</h1>

<p><a href="#news1">Jump to New content 1</a></p>
<p><a href="#news2">Jump to New content 2</a></p>

<p>Click on the links above and the :target selector highlight the current active HTML anchor.</p>

<p id="news1"><b>New content 1...</b></p>
<p id="news2"><b>New content 2...</b></p>
p:target {
border: 2px solid #D4D4D4;
background-color: #e5eecc;
}

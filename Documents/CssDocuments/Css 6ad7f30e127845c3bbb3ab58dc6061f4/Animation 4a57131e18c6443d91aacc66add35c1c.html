Animation

animation: Html elementinin tüm animasyon özelliklerini belirler

varsayılan değerler: none | 0 | ease | 0 | 1 | normal | none | running |

  • animation: Parametreleri
    • animation-name: Animasyona vericeğiniz benzersiz isim.
    • animation-duration Animasyon işlemi kaç saniye içinde tamamlanıcak ise süresi yazılır.
    • animation-timing-function Animasyonun ilerleme hızı (default:ease)
    • animation-delay Animasyon kaç süre sonra başlasın
    • animation-iteration-count Animasyonuz kaç kez tekrarlancağını gireriz.
    • animation-direction
    • animation-fill-mode Animasyon nerde dursun onu seçiyorsun.
    • animation-play-state Animasyon dursun mu çalışsın mı karar veriyorsun

animation-name:

animation-name: özelliği ile animasyona vericeğiniz benzersiz isimdir animasyonu bu isimle çağırır ve kullanırız. (varsayılan değer ⇒ none)

animation-duration:

animation-duration: özelliği ile animasyon ne kadar zaman içersinde tamamlansın onu bildiririz. Milisaniye veya saniye cinsinden değer verebiliriz → 2000ms veya 2s. (varsayılan değer ⇒ 0)

animation-timing-function:

animation-timing-function: özelliği ile animasyon geçiş hızını belirleriz.
  • parametreleri:
    • ease ;Yavaş olarak başlayarak hızlanır sona doğru yavaşlar. (varsayılan değer)
    • ease-in ;Yavaş başlar - sona doğru hızlı.
    • ease-out ;Hızlı başlar - sona doğru yavaş.
    • ease-in-out ;Yavaş başlar - ortada hızlı - sona doğru yavaş.
    • linear ;Sabit hızla başlar biter.
    • step-end ;Animasyon süresine bakılarak , geçiş süresi kadar bekleyerek doğrudan sona ulaşır.
    • step-start ;Animasyon süresine bakılmaksızın doğrudan sona ulaşır.
    • steps() ;Başlangıç ve bitiş arasında adımlamalı animasyon işlemi yapar.
    • cubic-bezier();bezier eğrisi (http://cubic-bezier.com) hızı kendin belirle.

animation-delay:

animation-delayözelliği ile animasyon ne kadar süre sonra başlasın onu belirtiyoruz.milisaniye veya saniye cinsinden değer girebiliriz → 2000ms veya 2s. (varsayılan değer ⇒ 0)

animation-iteration-count:

animation-iteration-count: özelliği ile animasyon kaç kez çalışsın onu belirtiyoruz.infinite veya sayı değerini alıyor. (varsayılan değer⇒ 1)
  • parametreleri:
    • infinite; Animasyon durmadan devam edicek.
    • sayı; Kaç kez tekrarlanmasını istiyorsan sayısını yaz.

animation-direction:

animation-direction: özelliği ile animasyon nasıl oynatılsın onu belirtiyoruz.

  • parametreleri:
    • normal ;normal oynat (varsayılan değer)
    • alternate ;alternatif (git gel yapıyor yani başa dönmüyor)
    • alternate-reverse ;ters alternatif (tersten git gel yapıyor yani başa dönmüyor)
    • reverse ;tersten oynat

animation-fill-mode:

animation-fill-mode: özelliği ile animasyonun hangi konumda duracağını belirtiriz.
  • parametreleri:
    • both ;her ikiside yani animasyonun nerede durucağına css karar verir.
    • forwards ;ileride bulunduğu son konumda durucak.
    • backwards ;geride bulunduğu ilk konumda durucak.
    • none ;değer yok (varsayılan değer)

animation-play-state:

animation-play-state: özelliği ile animasyonun çalışıp çalışmayacağını belirtiriz.
  • parametreleri:
    • running; animasyon çalışır. (varsayılan değer)
    • paused; animasyon duraklatlır.

@keyframes

@keyframes animasyonlarımızı oluşturacağımız yerdir iki farklı biçimde oluşturulabilir:
  • from - to kullanılarak iki aşamalı bir animasyon yapılabilir.
@keyframes animation-name{
from{

			color:black;	

	}

to{

			color:white;	

	}
}
  • %ler kullanılarak çok aşamalı bir animasyon yapılabilir.
@keyframes animation-name{
0%{

				color:black;	

	}

25%{

				color:blue;	

	}

50%{

				color:cyan;	

	}

75%{

				color:purple;	

	}

100%{

				color:green;	

	}
}

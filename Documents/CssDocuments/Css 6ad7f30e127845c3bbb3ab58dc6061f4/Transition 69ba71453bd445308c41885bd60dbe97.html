Transition

transition: Html elementinini tüm geçiş özelliklerini belirler.
  • transition: parametreleri:
    • transition-property
    • transition-duration
    • transition-timing-function
    • transition-delay
  • transition’ın varsayılan değerleri transition:all 0 ease 0;
  • hepsini yan yana kullanma taktiği parametre sırası transition: all 900ms linear 1s;

transition-property:

transition-property: all veya css özellik adı yani bu işlem hangi özelliğe uygulansın onu belirtiyoruz.
.deneme{
    width: 50%;
    height: 10vh;
    border: #0008ff solid 5px;
    transition-property:color;
}
.deneme:hover{
		color:black;
    transform-origin:center ;
    transform: rotateY(180deg);
}

transition-duration:

transition-duration: Geçiş süresi işlem kaç saniyede tamamlansın onu belirtiyoruz. Saniye veya milisaniye cinsinden değer girebiliriz → 2000ms veyahut 2s.
.deneme{
    width: 50%;
    height: 10vh;
    border: #0008ff solid 5px;    
    transition-property:color;    /* değerlerin uygulanacağı css adı */
		transition-duration:350ms;    /* geçiş işlem süresi */
}
.deneme:hover{
		color:black;
    transform-origin:center ;
    transform: rotateY(180deg);
}

transition-timing-function:

transition-timing-function: Hız eğrisi geçiş ne hızda ilerlesin onu belirtiyoruz.
  • parametreler:
    • ease ; varsayılan değerdir yavaş başlayıp ortada hızlanır sona doğru yavaşlar.
    • linear ; hız her eğride sabittir geçiş sabit hızda ilerler.
    • step-start ;Animasyon süresine bakılmaksızın doğrudan sona ulaşır.
    • step-end ;Geçiş süresine bakılarak , geçiş süresi kadar bekleyerek doğrudan sona ulaşır.
    • cubic-bezier() ;
    • örnek: transition-timing-function:cubic-bezier(0,1,1,0) ; hızlı başlasın ortada beklesin sonra birden hızlanıp bitirsin.
    • http://cubic-bezier.com bu siteden geçiş hızını kendin belirleyebilirsin.
.deneme{
    width: 50%;
    height: 10vh;
    border: #0008ff solid 5px;
    transition-property:color;                        /* değerlerin uygulanacağı css adı */
    transition-timing-function:cubic-bezier(0,1,1,0); /* Hız eğrisi */
		transition-duration:350ms;                        /* geçiş işlem süresi */
}
.deneme:hover{
		color:black;
    transform-origin:center ;
    transform: rotateY(180deg);
}

transition-delay:

transition-delay:; Geçiş ne kadar süre sonra başlıyacağını belirtiriz.
  • varsayılan değeri 0’dır. saniye veya milisaniye cinsinden değer girebiliriz.
  • /verdiğin süre kadar bekleyip sonra başlıyor, gecikmeli başlıyor/
.deneme{
    width: 50%;
    height: 10vh;
    border: #0008ff solid 5px;    
    transition-property:color;    /* değerlerin uygulanacağı css adı */
		transition-duration:350ms;    /* geçiş işlem süresi */
    transition-timing-function:cubic-bezier(0,1,1,0); /* Hız eğrisi */
		transition-delay:2s;          /* geçiş 2 saniye bekledikten sonra başlar */
}
.deneme:hover{
		color:black;
    transform-origin:center ;
    transform: rotateY(180deg);
}
